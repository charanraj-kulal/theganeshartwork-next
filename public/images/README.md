# Image Assets

This directory contains all image assets for the Decorior clone website.

## Structure

- `/products` - Product images
- `/categories` - Category images
- `/slider` - Hero slider images
- `logo.png` - Main logo
- `placeholder.jpg` - Placeholder for missing images

## Note

Replace these placeholder paths with actual images from the decorior.in website or use your own images.

To get images from the original site, you can:
1. Visit https://decorior.in/
2. Right-click on images and save them
3. Place them in the appropriate folders
4. Update image paths if needed
